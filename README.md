# CustomTemplate

This repository is a reusable .NET solution template. It includes multiple projects: API, Blazor (Portal), Worker Service, Business, Library, Console, and Database.

## Use as a dotnet template

See `INSTALL_TEMPLATE.md` for step-by-step instructions to install this repository as a local template and generate new solutions from it.

Quick start:

```powershell
# From the repo root
dotnet new install .
dotnet new custom-sln -n MyNewSolution
```

For more details, parameters, and troubleshooting, open `INSTALL_TEMPLATE.md`.
